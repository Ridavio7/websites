## Результат работы ping

Строка запроса:

 `http://<ip>/api/getVersion.api`

Метод:

`GET`

**Пример:**

* *request:*

`http://192.168.4.1/api/getVersion.api`

содержимое файла передается в поле данных запроса

* *response:*

```
{
  "name": "DT_ETH_TEST_01_COM",
  "versions": {
    "software": 6,
    "hardware": 1
  }
}
```
