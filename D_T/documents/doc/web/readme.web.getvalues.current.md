## Результат работы ping

Строка запроса:

 `http://<ip>/api/getValues.api`

Метод:

`GET`

**Пример:**

* *request:*

`http://192.168.4.1/api/getValues.api`

содержимое файла передается в поле данных запроса

* *response:*

