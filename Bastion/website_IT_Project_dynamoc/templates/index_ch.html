Avantguard
实现特点
在线技术平台自动向移动和固定设备显示分析数据 随后提供自动或手动模式的响应选项。 该平台部署在一台或多台服务器上、 既可以是物理机，也可以是虚拟机。技术平台在 Astra Linux 操作系统的控制下运行。 也可以使用其他 Linux、FreeBSD 和 Windows 发行版来部署和使用平台组件。系统结构允许将 在不同的服务器上安装不同的服务， 从而提供可扩展性、可控的计算负荷分配和网络接口负载。系统设有服务冗余机制，当某些资源不可用时，系统会自动重建信息流。
实施的效果
  1. 可以拒绝远程监护服务，只支付响应费用。
  2. 通过该平台，可以实时接收有关设施运行情况的可靠信息、 从而消除对外部承包商和员工的依赖。
  3. 能够从世界上任何有互联网的地方远程访问、响应、管理和分析信息。
  4. 所有设备和平台访问都可以托管在客户的场所，也可以作为云服务提供。
  5. 这些设备可以方便地运输并连接到其他地点。
  6. 监控连接到技术平台的设备和系统的状态。
  7. 最大限度地降低工艺维护和保养的人工成本。
