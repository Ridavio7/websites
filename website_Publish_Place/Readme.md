# Publish place
## A private site for reading and publishing original articles  
#### The site features:
- registration;
- authentication;
- preview of published articles;
- full preview of published articles;
- publication of articles.
#### Software part of the website:
- Beakend: golang;
- Frontend: HTML, CSS;
- Db: MySQL.
